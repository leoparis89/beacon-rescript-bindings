# `Beacon SDK Rescript Bindings`

## Status

⚠️ Work in progress

Only WalletClient currently implemented

## Installation

```console
yarn add rescript-beacon-sdk
```

`rescript-beacon-sdk` should be added to `bs-dependencies` in your
`bsconfig.json`:

```diff
{
  //...
  "bs-dependencies": [
    // ...
+    "rescript-beacon-sdk"
  ],
  //...
}
```
